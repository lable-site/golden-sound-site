# Golden Sound — Официальный сайт

## Структура файлов
```
/
├── index.html          ← Главная страница (HTML-скелет)
├── logo.png            ← Логотип (можно заменить своим)
├── css/
│   └── styles.css      ← ВСЕ стили сайта (золотая тема)
└── js/
    ├── main.js         ← Точка входа, порядок загрузки
    ├── config.js       ← Ключи Supabase + резервные артисты
    ├── content.js      ← Данные из Supabase: услуги, статистика, соцсети, контакты
    ├── artists.js      ← Слайдер артистов (НЕ ТРОГАТЬ настройки Swiper!)
    ├── canvas.js       ← Частицы и звёзды (золотые)
    └── animations.js   ← Reveal-анимации и счётчики
```

## Переключение режимов
В файле `js/config.js`:
- `USE_MOCK = false` → данные из Supabase (продакшн)
- `USE_MOCK = true`  → данные из резервных заглушек (тест без интернета)

## Контакты в Supabase
Если хочешь чтобы адреса/телефон/email брались из базы — добавь в таблицу
`site_config` строки с ключами: `address_1`, `address_2`, `phone`, `email`.
Пока этих ключей нет — показываются Golden Sound контакты по умолчанию.
